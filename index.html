Introduction

The Open Web UI ( with Payment Integration ) is a flexible, open-source platform built on top of the original Open_WebUI. This project extends the capabilities of the base framework by incorporating a subscription system, enabling seamless integration with services like Stripe for payment processing. Designed for ease of customization and versatility, this module supports various backend configurations, databases, and deployment options, making it suitable for a wide range of applications. This documentation will guide you through setting up, developing, testing, and deploying the platform efficiently.

Development

Frontend

The frontend of Open Web UI is built using modern web technologies such as:

  • Svelte for building dynamic user interfaces.
  • TailwindCSS for styling.
  • Integration with RESTful APIs for data communication.

See the Frontend Development Guide for details on setting up your environment and extending the UI.

Backend

The backend provides the core business logic and API services, built using:

  • FastAPI for handling API requests.
  • PostgreSQL as the default database for managing data.

Refer to the Backend Development Guide for more details on API development and extending functionality.

Database

We use **PostgreSQL** as the primary database for Open Web UI. The database structure is designed for scalability and performance. For advanced users, there are options to integrate other database systems.

Learn more about configuring and managing the database in the Database Guide.
